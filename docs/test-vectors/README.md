# AICPM Test Vectors

This directory contains **canonical test vectors** for validating
AICPM (Authenticated AI Content Provenance Marking) implementations.

Each test vector consists of:
- A sample AICPM manifest
- An expected verification outcome

These vectors are intended for:
- Verifier implementers
- Interoperability testing
- Regression testing
- Standards review

All cryptographic material in this directory is **test-only**.
